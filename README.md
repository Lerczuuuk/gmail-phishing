# Gmail Phishing Simulator

> ⚠️ **Disclaimer**: This project is for educational purposes only. Do not use it maliciously.

## About

This is my first Python project where I practice object-oriented programming. The program simulates a Gmail login form with:
- Email and password validation.
- Logging valid and invalid login attempts into separate files.
- Redirecting users to the actual Gmail website.

---

## How to Run

### Requirements:
- **Python 3.10+**
- Install dependencies:
  ```bash
  pip install pillow customtkinter
  ```

### Steps:
1. Ensure you have all the files (including `Google.png`) in one folder.
2. Run the program:
   ```bash
   python main.py
   ```
3. Use the GUI to test login validation. Logs are saved in `logins.valid.txt` and `logins.invalid.txt`.
